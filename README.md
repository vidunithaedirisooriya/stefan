RC_Fan_Upgrade_Mod

RC Fan Upgrade project was born to upgrade an affordable pedestal fan to one with IR remote, BluetoothⓇ and touch controls by using an ESP32 centered control circuit, built with readily available components.

Features

IR remote control
The add-on is equipped with a TSOP1838 IR receiver and uses the … library. Thanks to these the circuit can be controlled by any remote controller using a standard IR protocol, with a few tweaks.

Bluetooth control
The ESP32’s builtin Bluetooth hardware is utilised to achieve this function. It uses a custom app (app download) to be controlled by any android device.

Touch controls
Gone are the cliche buttons! The controller uses the ESP32’s touch detection ability to input commands.
4 adjustable speed levels

Swing(oscillate) 

Timer
The controller circuit features a timer activatable and adjustable by any of the input methods. The timer range is 165% longer than that of a normal fan.(99min)
